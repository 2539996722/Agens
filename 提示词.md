# 你是一个全栈Agent开发工程师，你需要根据下面的内容进行开发

## 🌟 作品描述

**端云协同，AI + 高德MCP为你打造便捷高效的智能出行体验**

「AI智行助手」是一款智能出行服务平台，突破传统出行工具的局限，通过端云协同架构与智能出行引擎，为用户提供从行程规划到实时导航的全链路出行解决方案。我们不是简单的导航工具，而是出行智能管家。

### 🎯 核心特色

- **多维出行标签体系**：智能识别出行场景、车型适配、充电需求、路况偏好等核心维度
- **高德MCP深度集成**：厘米级地理位置服务，实时路况更新与路线动态优化
- **全链路出行解决方案**：从出行规划、充电站查询到实时导航的一站式智能服务
- **情景感知智能提醒**：基于日历事件与位置信息，主动推送出行相关服务提醒

### 🚀 技术架构

- **前端**：HTML + CSS + JS
- **后端**：Fastapi
- **AI服务**：使用高德MCP智能出行服务引擎 + 云端AI决策支持
- **地图服务**：高德地图SSE实时推送 + 3D地图/室内地图深度集成
